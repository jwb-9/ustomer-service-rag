# 客服知识库

## 项目简介

本项目是一个基于 RAG（Retrieval-Augmented Generation）技术的客服知识库问答系统，用于帮助客服、售前顾问和实施顾问快速准确地回答用户关于套餐、额度、发票、退款、数据保留、团队协作和企业版支持范围的问题。

## 技术栈

- **向量数据库**: Milvus
- **嵌入模型**: BGE-M3（1024维）
- **语言模型**: GPT-5.4-mini
- **框架**: LangChain
- **编程语言**: Python

## 功能特点

1. **知识库管理**: 支持文档上传、切分和向量化存储
2. **智能检索**: 基于向量相似度的快速检索
3. **问答生成**: 结合检索结果生成精准回答
4. **多套餐支持**: 覆盖试用版、基础版、专业版、企业版
